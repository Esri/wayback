World Imagery Wayback

WHAT

Wayback imagery is a digital archive of the World Imagery basemap, enabling users to access different versions of World Imagery captured over the years. Each record in the archive represents World Imagery as it existed on the date new imagery was published. Wayback currently supports all updated versions of World Imagery dating back to February 20, 2014. There is an AGOL item for every version and each of these items can be viewed in the Wayback Imagery group.

WHY

The World Imagery basemap is regularly updated with more current imagery. When and where updates occur, the previous imagery is replaced and is no longer visible. For many use cases, the updated imagery is more desirable and typically preferred. Other times, however, the previous imagery may support use cases that the new imagery does not. In these cases, a user may need to access a previous version of the World Imagery basemap.

HOW

Previous World Imagery basemap updates are presented within a timeline and as a list. Updates that resulted in local (to the current map view) changes are highlighted with a white band. Choose any of these updates to load the basemap with that vintage. One or more updates can be added to an export queue and pushed to a new ArcGIS Online web map.
